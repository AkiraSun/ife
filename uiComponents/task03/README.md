# 图片裁剪及上传

## 如何运行
```bash
npm i
```

安装依赖完成后，就可以运行项目

```bash
npm start
```

打开浏览器网址 `http://127.0.0.1:3000`可以看到图片上传按钮，点击上传表单后，调整裁剪框位置、大小后，点击”上传图片到服务器“即可在项目根目录生成临时图片文件。
> 如果提示权限问题，可以使用`sudo npm start`开启服务后再尝试。
